## 논의 사항

1. **Human vs AI 불공정 버그 (BUG-UI-001)**: 애벌레가 직접 게임 플레이 중 발견. Human이 여러 그룹을 동시에 배치할 수 없었던 반면 AI는 API로 여러 그룹 동시 제출 가능 → 수정 완료.
2. **Google OAuth 재발 방지**: values.yaml에서 키 자체를 제거하는 방식으로 ArgoCD selfHeal 충돌 근본 해소. inject-secrets.sh는 클러스터 초기 구성 시 1회 실행.
3. **테스트 신뢰성**: 중간 과정에서 K8s 구버전 배포 상태로 테스트가 반복 실패하는 상황 발생. 코드 수정 후 반드시 `docker build + rollout restart` 선행 필요 — 절차 확립.
4. **repository 커버리지 0%**: 현재 스프린트 블로커 아님. Sprint 5에서 인터페이스 모킹 기반 테스트 도입 검토.
