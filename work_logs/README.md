# 프로젝트 로그 (Project Logs)

개발 과정의 모든 기록을 남기는 공간.

## 디렉토리 구조

```
logs/
├── sessions/         세션 로그 (작업 세션별 기록)
├── daily/            데일리 로그 (일일 스탠드업/진행 기록)
├── vibe/             바이브 로그 (아이디어, 영감, 자유 메모)
├── retrospectives/   회고 로그 (Sprint 회고)
└── decisions/        결정 로그 (기술 결정, 트레이드오프 기록)
```

## 파일 명명 규칙

| 로그 유형 | 파일명 형식 | 예시 |
|-----------|-------------|------|
| 세션 로그 | `YYYY-MM-DD-NN.md` | `2026-03-08-01.md` |
| 데일리 로그 | `YYYY-MM-DD.md` | `2026-03-08.md` |
| 바이브 로그 | `YYYY-MM-DD-제목.md` | `2026-03-08-ai-심리전-아이디어.md` |
| 회고 로그 | `sprint-NN.md` | `sprint-01.md` |
| 결정 로그 | `DNN-제목.md` | `D01-백엔드-언어-결정.md` |
