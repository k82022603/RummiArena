오늘 추가된 **새 그룹 드롭존 UI** (NewGroupDropZone) 를 활용해 주세요:

1. 타일을 드래그하기 시작하면 보드 우하단에 **점선 "+ 새 그룹"** 영역이 나타납니다
2. 기존 블록과 상관없이 **독립된 새 그룹**을 만들려면 이 영역에 드롭
3. 기존 블록을 확장(런 연속, 그룹 합류)하려면 해당 블록 **위에 직접 드롭**

- 빈 보드 공간에 드롭 시 인접 서버 확정 그룹으로 잘못 매핑될 수 있습니다. 서버가 거절하면 UI 가 되돌려 놓으므로 **게임은 계속 가능**하지만, 번거로우면 위 방법대로 "+ 새 그룹" 버튼을 적극 활용하세요
