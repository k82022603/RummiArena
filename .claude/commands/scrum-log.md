스크럼 미팅 로그를 작성한다.

## 규칙
- 경로: `work_logs/scrums/YYYY-MM-DD-NN.md`
- 하루 여러 개 가능. NN은 같은 날짜 내 순번 (01, 02, ...)

## 실행
1. 같은 날짜의 기존 스크럼 파일 확인하여 다음 순번 결정
2. 대화 컨텍스트에서 공유 내용, 논의 사항 정리
3. 액션 아이템 도출
4. 사용자에게 검토 요청

## 템플릿
```markdown
# 스크럼 미팅 로그

- **날짜**: YYYY-MM-DD
- **Sprint**: Sprint N
- **참석자**: 애벌레, Claude

## 각자 공유 (3가지)

### 애벌레
- **어제**:
- **오늘**:
- **블로커**:

### Claude
- **어제**:
- **오늘**:
- **블로커**:

## 논의 사항
-

## 액션 아이템
| 담당 | 할 일 | 기한 |
|------|-------|------|
| | | |

## 메모
-
```

## 주의사항
- 톤: 대화체
- 참석자 기본값: 애벌레 + Claude
- 대화에서 논의된 내용을 기반으로 채우기
