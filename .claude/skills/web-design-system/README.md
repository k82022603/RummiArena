# 웹 디자인 시스템 (Web Design System)

> Verbalized Sampling 기법으로 제약 기반 디자인을 생성한다.

## 언제 사용하나

- UI 컴포넌트 신설 또는 리디자인 시
- 색상/타이포그래피/레이아웃 결정 시
- designer/frontend-dev 협업 시 디자인 표준 참조

## 핵심 흐름

1. **Anti-Pattern 식별**: 기존 디자인의 문제점 탐색
2. **제약 적용**: TailwindCSS 색상 팔레트 + 타이포 스케일 + 간격 체계
3. **Verbalized Sampling**: 자연어로 디자인 의도를 명시한 후 코드로 변환
4. **컴포넌트 패턴**: 카드/모달/폼/테이블/네비게이션 표준
