### UAT-01 | 로그인 및 초기 화면

**목적**: 로그인 후 로비 진입 확인

**단계**:

1. 브라우저에서 `http://localhost:30000` 접속
2. 로그인 페이지가 표시되는지 확인
3. 닉네임 입력란에 원하는 이름 입력 (예: `테스터1`)
4. "게스트로 로그인" 버튼 클릭
5. 로비 화면으로 이동되는지 확인

**기대 결과**:

| 항목 | 기대 동작 |
|------|----------|
| 로그인 페이지 | 닉네임 입력란과 로그인 버튼이 보임 |
| 로그인 후 | 로비(`/lobby`)로 이동 |
| 상단 헤더 | 닉네임이 표시됨 |
| 메뉴 | 로비, 연습 모드, 랭킹 링크가 보임 |

**체크리스트**:
- [ ] 로그인 페이지 정상 로드
- [ ] 닉네임 입력 가능
- [ ] 로그인 버튼 클릭 후 로비로 이동
- [ ] 헤더에 내 닉네임 표시

### UAT-02 | 연습 모드 — Stage 1: 그룹 만들기

**목적**: 같은 숫자 다른 색상 3개를 그룹으로 만들어 클리어

**룰 설명**: 같은 숫자, 서로 다른 3~4가지 색상의 타일을 모으면 그룹.
예) 빨강 7 + 파랑 7 + 노랑 7 = 유효한 그룹 ✓

**단계**:

1. 상단 메뉴에서 "연습 모드" 클릭
2. Stage 1 카드 클릭
3. 튜토리얼 화면이 나타나면 "시작하기" 클릭
4. 하단 랙(패)에서 **같은 숫자, 다른 색상** 타일 3개를 찾아 보드로 드래그
   - 드래그 방법: 타일을 클릭한 채로 화면 중앙 보드 영역으로 끌어다 놓기
5. 오른쪽 상단에 "클리어 가능!" 배지가 나타나는지 확인
6. "클리어 확정!" 버튼 클릭

**기대 결과**:

| 항목 | 기대 동작 |
|------|----------|
| 드래그 | 타일이 보드에 배치됨 |
| 3개 같은 숫자 다른 색상 | 그룹 타입 패널에 "그룹" 표시 |
| 클리어 조건 달성 | "클리어 가능!" 배지 표시 |
| 확정 클릭 | 점수 및 결과 화면 표시 |

**체크리스트**:
- [ ] 튜토리얼 화면 정상 표시
- [ ] 타일 드래그 가능
- [ ] 유효한 그룹 배치 시 "클리어 가능!" 표시
- [ ] 무효 배치 시 에러 메시지 표시 (빨간 테두리)
- [ ] 초기화 버튼으로 원상복귀
- [ ] 확정 버튼 클릭 후 결과 화면

---

### UAT-03 | 연습 모드 — Stage 2: 런 만들기

**목적**: 같은 색상 연속 숫자 3개로 런 만들기

**룰 설명**: 같은 색상, 연속된 숫자 3개 이상이면 런.
예) 빨강 4 + 빨강 5 + 빨강 6 = 유효한 런 ✓

**단계**:

1. 연습 모드에서 Stage 2 선택
2. 튜토리얼 확인 후 "시작하기"
3. 하단 랙에서 **같은 색상, 연속된 숫자** 타일 3개를 보드에 드래그
4. 보드 아래 타입 버튼이 "런"으로 표시되는지 확인
5. "클리어 가능!" 확인 후 확정

**무효 케이스 테스트** (선택):
- 다른 색상 타일 3개 배치 → 에러 표시 확인
- 숫자 건너뛴 배치 (예: 4, 6만) → 에러 표시 확인

**체크리스트**:
- [ ] 연속 숫자 같은 색상 → "클리어 가능!" 표시
- [ ] 색상 혼합 시 에러 메시지
- [ ] 타입 버튼에 "런" 표시됨

---

### UAT-04 | 연습 모드 — Stage 3: 조커 활용

**목적**: 조커(JK)를 이용해 빠진 타일을 대체하는 세트 만들기

**룰 설명**: 조커는 어떤 타일로도 대체 가능.
예) 조커 + 빨강 5 + 빨강 6 = 조커가 빨강 4 또는 7 역할 ✓

**단계**:

1. Stage 3 선택
2. 랙에서 조커(JK) 타일을 찾아 보드에 드래그
3. 연속되는 색상 타일 2개를 더 드래그
4. "클리어 가능!" 확인 후 확정

**체크리스트**:
- [ ] 조커 타일 보드 배치 가능
- [ ] 조커 + 일반 타일 2개 → 유효한 세트 인식
- [ ] 조커만 배치 시 클리어 불가

### UAT-05 | 방 생성 및 AI 게임 시작

**목적**: 방을 만들고 AI 플레이어와 게임 시작

**단계**:

1. 로비(`/lobby`)에서 "방 만들기" 버튼 클릭
2. 방 설정 화면에서:
   - 방 이름 입력
   - 최대 인원 선택 (2~4명)
   - AI 플레이어 추가 (슬롯 클릭)
3. "방 만들기" 확인 클릭
4. 방 대기실에서 AI 슬롯이 채워졌는지 확인
5. "게임 시작" 버튼 클릭
6. 게임 화면으로 전환되는지 확인

**기대 결과**:

| 항목 | 기대 동작 |
|------|----------|
| 방 생성 | 방 대기실로 이동 |
| AI 슬롯 | AI 플레이어 이름과 캐릭터 표시 |
| 게임 시작 | 각 플레이어에게 타일 14장 배분 |
| 게임 화면 | 내 타일 랙, 게임 보드, 상대방 정보 표시 |

**체크리스트**:
- [ ] 방 생성 정상 동작
- [ ] AI 슬롯 추가 가능
- [ ] 게임 시작 버튼 활성화
- [ ] 게임 화면 정상 진입
- [ ] 내 타일 14개 표시

### UAT-06 | 게임 플레이 — 타일 배치 및 확정

**목적**: 실제 게임에서 타일을 배치하고 유효성 검사 후 확정

**단계**:

1. 내 턴이 돌아오면 랙 강조 표시 확인
2. 랙에서 유효한 세트(그룹 또는 런) 타일을 보드로 드래그
3. 보드에 배치된 타일 그룹의 유효성 확인
   - 유효: 초록 테두리
   - 무효: 빨간 테두리 + 에러 메시지
4. 유효한 배치 후 "확정" 버튼 클릭
5. 다음 플레이어 턴으로 넘어가는지 확인

**초기 등록 규칙 (첫 배치)**:
> 첫 턴에는 내 타일만으로 30점 이상을 배치해야 함. 10짜리 3개 = 30점 ✓

**체크리스트**:
- [ ] 내 턴에만 드래그 가능 (상대 턴에는 비활성)
- [ ] 유효한 배치 → 확정 버튼 활성
- [ ] 무효 배치 → 확정 버튼 비활성
- [ ] 확정 후 턴 이동 확인
- [ ] 보드에서 랙으로 타일 되돌리기 가능

---

### UAT-07 | 게임 플레이 — 드로우 및 턴 넘기기

**목적**: 배치 없이 타일을 가져오거나 턴 넘기기

**단계**:

1. 내 턴에 아무 타일도 배치하지 않고 "드로우" 버튼 클릭
2. 랙에 타일 1장이 추가되는지 확인
3. 상대방 턴으로 넘어가는지 확인

**체크리스트**:
- [ ] 드로우 버튼 클릭 시 랙에 타일 추가
- [ ] 드로우 후 내 턴 종료
- [ ] 보드에 타일 배치 중 "되돌리기" 클릭 시 원상복귀

---

### UAT-08 | 게임 종료 및 결과 화면

**목적**: 누군가 패를 다 내면 게임 종료 및 결과 확인

**확인 항목**:

1. 플레이어 중 패가 0장이 되면 게임 종료 팝업 표시
2. 승자/패자 결과 화면 표시
3. 각 플레이어의 남은 타일 점수 합산 표시
4. ELO 점수 변동 표시

**체크리스트**:
- [ ] 게임 종료 시 결과 오버레이 표시
- [ ] 승자 강조 표시
- [ ] 점수 합산 결과 표시
- [ ] "다시 하기" 또는 "로비로" 버튼 동작

---

### UAT-09 | 랭킹 페이지 조회

**목적**: 플레이어 ELO 랭킹 확인

**단계**:

1. 메뉴에서 "랭킹" 클릭 (`/rankings`)
2. 리더보드에 플레이어 목록 표시 확인
3. 티어별 탭 (브론즈/실버/골드/플래티넘 등) 필터 확인
4. 내 닉네임 순위 확인

**체크리스트**:
- [ ] 랭킹 페이지 정상 로드
- [ ] 플레이어 목록 표시 (ELO 점수 포함)
- [ ] 티어 필터 동작
- [ ] 내 순위 강조 표시

### UAT-10 | 연습 모드 — Stage 4~6 (심화)

**목적**: 복합 배치, 마스터 모드 클리어

| Stage | 클리어 조건 | 난이도 |

**체크리스트**:
- [ ] Stage 4 런+그룹 복합 배치 클리어
- [ ] Stage 5 다중 그룹 배치 클리어
- [ ] Stage 6 12장 이상 배치 클리어
