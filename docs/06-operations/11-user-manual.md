# RummiArena 사용자 매뉴얼

- **대상**: 게임을 처음 시작하는 사용자
- **작성일**: 2026-05-10

---

## RummiArena란?

RummiArena는 루미큐브(Rummikub) 보드게임을 온라인으로 즐길 수 있는 플랫폼입니다.
사람끼리 대전하거나, GPT·Claude·DeepSeek·Ollama 같은 AI와 함께 플레이할 수 있습니다.

---

## 1. 시작하기

### 1.1 접속

브라우저를 열고 주소창에 입력합니다:

```
http://localhost:30000
```

### 1.2 로그인

**Google 계정 로그인:**
1. "Google로 로그인" 버튼 클릭
2. Google 계정 선택
3. 권한 허용

로그인 후 우측 상단에 내 이름이 표시되면 완료입니다.

### 1.3 프로필 설정

우측 상단 아바타 클릭 → 프로필 설정:
- **닉네임**: 게임에서 표시되는 이름
- **아바타**: 프로필 이미지

---

## 2. 방 만들기와 참가

### 2.1 방 목록 화면

로비에서 현재 열려있는 방 목록을 볼 수 있습니다.
각 방에는 참가 인원, 방 이름, 상태(대기 중 / 게임 중)가 표시됩니다.

### 2.2 새 방 만들기

"방 만들기" 버튼 클릭 후 설정:

| 항목 | 설명 |
|------|------|
| 방 이름 | 원하는 이름 입력 |
| 최대 인원 | 2~4명 선택 |
| AI 플레이어 추가 | 원하는 경우 AI 추가 (아래 설명 참조) |

**AI 플레이어 추가하기:**
- 모델 선택: GPT / Claude / DeepSeek / Ollama(LLaMA) 중 선택
  - GPT, Claude, DeepSeek는 클라우드 AI (빠름, 유료)
  - Ollama는 로컬 AI (느림—약 25초, 무료)
- 캐릭터 선택: 6종 중 선택 (아래 캐릭터 설명 참조)
- 난이도: 하수 / 중수 / 고수

### 2.3 방 참가하기

방 목록에서 원하는 방 클릭 → "참가" 버튼.

대기 중(`WAITING`) 상태의 방에만 참가할 수 있습니다.
게임이 시작된 방(`PLAYING`)에는 참가할 수 없습니다.

## 3. 루미큐브 게임 방법

### 3.1 기본 규칙

루미큐브는 숫자 타일을 조합해 필드에 내려놓는 게임입니다.

**타일 구성:**
- 1~13 숫자 × 4가지 색깔(빨강, 파랑, 노랑, 검정) × 2세트 = 104장
- 조커 2장 (어떤 타일로도 사용 가능)
- 총 106장

**타일 조합 규칙 (2가지):**

| 조합 | 설명 | 예시 |
|------|------|------|
| **그룹 (Group)** | 같은 숫자, 다른 색깔 3~4개 | 빨강7 + 파랑7 + 노랑7 |
| **런 (Run)** | 같은 색깔, 연속 숫자 3개 이상 | 빨강4 + 빨강5 + 빨강6 |

### 3.2 게임 진행 순서

1. **타일 배분**: 각 플레이어에게 14장씩 배분
2. **첫 배치 (초기 용융)**: 처음 필드에 타일을 올리려면 합계 **30점 이상**이어야 함
3. **내 턴이 되면**: 타일을 배치하거나, 배치할 수 없으면 드로우(1장 가져오기)
4. **확정 버튼**: 배치가 완료되면 확정 버튼을 눌러 턴 종료
5. **승리 조건**: 가장 먼저 손패의 타일을 모두 필드에 내려놓은 사람이 승리

### 3.3 재배치 규칙

이미 필드에 나와 있는 타일도 재조합할 수 있습니다.
예: 빨강4+5+6 런에서 5를 빼서 다른 조합에 쓸 수 있음.
단, **턴 종료 시 필드 위의 모든 조합이 유효해야** 합니다.

유효하지 않은 상태로 확정을 누르면 오류가 발생합니다.
그 경우 타일을 원래대로 되돌리거나, 올바른 조합을 만들어야 합니다.

### 3.4 조커 사용법

조커는 어떤 타일로도 대체할 수 있습니다.
- 조커를 원하는 위치에 드래그해 배치
- 이미 필드에 있는 조커를 해당 조커가 대체하는 타일과 교환 가능 (그 타일이 내 손패에 있을 때)

## 4. 화면 구성 안내

### 4.1 게임 화면 레이아웃

```
┌─────────────────────────────────────────┐
│  상단: 플레이어 정보 + 타이머            │
├─────────────────────────────────────────┤
│                                         │
│  메인 보드 (필드)                        │
│  — 여기에 타일 조합을 올림               │
│                                         │
├─────────────────────────────────────────┤
│  하단: 내 손패 (My Rack)                │
│  [확정] [드로우] [되돌리기] 버튼          │
└─────────────────────────────────────────┘
```

### 4.2 상단 플레이어 정보

- 각 플레이어의 이름, 손패 개수, ELO 점수 표시
- 현재 차례인 플레이어 강조 표시
- 남은 시간 타이머 (시간 내 액션 없으면 강제 드로우)

### 4.3 버튼 설명

| 버튼 | 기능 |
|------|------|
| **확정** | 현재 배치 상태로 턴 종료 (유효성 검사 후 적용) |
| **드로우** | 타일 1장 가져오기 (배치 없이 턴 종료) |
| **되돌리기** | 이번 턴에 한 배치 전부 취소하고 처음 상태로 |

**확정 버튼이 비활성화될 때**: 현재 필드 배치가 유효하지 않거나, 내 턴이 아닐 때. 필드 타일을 올바른 조합으로 만들어야 활성화됩니다.

## 5. AI 플레이어 이해하기

### 5.1 AI 캐릭터 6종

| 캐릭터 | 스타일 | 특징 |
|--------|--------|------|
| **Rookie** | 보수적 | 안전한 수만 선택. 큰 실수 없음 |
| **Calculator** | 확률 기반 | 기대값 계산으로 최적 배치. 안정적 |
| **Shark** | 공격적 | 상대 견제 + 대량 배치. 폭발력 있음 |
| **Fox** | 기만적 | 의도적 지연 + 역전 노림 |
| **Wall** | 수비적 | 최소 배치 + 자원 비축. 중반 이후 강함 |
| **Wildcard** | 예측 불가 | 랜덤 전략 혼합. 상대하기 어려움 |

### 5.2 AI 턴 대기 시간

AI가 생각하는 시간이 있습니다:
- **GPT, DeepSeek**: 10~120초 (추론 모델 사용)
- **Claude**: 30~120초 (thinking 모드)
- **Ollama (LLaMA)**: 20~60초 (CPU 추론)

AI 턴 중에는 상단에 "AI가 생각 중입니다..." 표시가 나타납니다. 응답이 오면 자동으로 진행됩니다.

**Ollama AI는 왜 느린가요?**
Ollama는 클라우드 서버가 아닌 로컬 컴퓨터에서 AI를 실행합니다. GPU 없이 CPU만 사용하기 때문에 응답이 느립니다. 무료라는 장점이 있습니다.

## 6. ELO 랭킹 시스템

### 6.1 ELO 점수란?

체스에서 사용되는 실력 점수 체계입니다. 이길 때마다 올라가고, 질 때마다 내려갑니다.
상대가 강할수록 이겼을 때 점수가 많이 오릅니다.

### 6.2 티어 시스템

| 티어 | ELO 범위 | 색상 |
|------|----------|------|
| 브론즈 | ~ 1199 | 갈색 |
| 실버 | 1200 ~ 1399 | 회색 |
| 골드 | 1400 ~ 1599 | 금색 |
| 플래티넘 | 1600 ~ 1799 | 하늘색 |
| 다이아 | 1800 ~ 1999 | 파랑 |
| 마스터 | 2000 ~ | 보라 |

초기 점수는 1200점(실버)에서 시작합니다.

### 6.3 랭킹 확인

로비 화면 우측 또는 상단 메뉴에서 "랭킹" 탭을 클릭합니다.
전체 랭킹 / 티어별 랭킹을 볼 수 있습니다.

## 7. 자주 묻는 질문 (FAQ)

**Q: 게임이 멈춘 것 같아요.**
A: AI 차례 중일 가능성이 높습니다. 상단의 타이머와 "AI가 생각 중" 메시지를 확인하세요. 특히 DeepSeek V4-Pro나 Claude는 복잡한 판에서 1~2분 걸릴 수 있습니다. 최대 대기 시간(1000초)이 지나면 AI가 자동으로 드로우합니다.

**Q: 확정 버튼이 눌리지 않아요.**
A: 두 가지 경우가 있습니다.
- 필드의 타일 조합이 유효하지 않을 때 (빨간색 표시됨) → 올바른 조합으로 수정
- 내 턴이 아닐 때 → 상단에서 현재 누구 차례인지 확인

**Q: 연결이 끊어졌어요.**
A: 브라우저를 새로고침하면 자동으로 재연결됩니다. 게임 상태는 서버에 저장되어 있어 진행 상황이 유지됩니다.

**Q: AI가 계속 드로우만 해요.**
A: AI가 배치 가능한 타일 조합을 찾지 못한 경우입니다. 특히 게임 초반에 자주 발생합니다. 정상 동작입니다.

**Q: Ollama AI를 선택했는데 너무 느려요.**
A: Ollama는 로컬 CPU로 AI를 실행하기 때문에 응답에 25~60초 걸립니다. 빠른 게임을 원하신다면 GPT나 DeepSeek를 선택하세요.

**Q: 연습 모드가 있나요?**
A: 있습니다. 로비에서 "연습 모드" 버튼을 클릭하면 결과가 ELO에 반영되지 않는 연습 게임을 즐길 수 있습니다.

**Q: 조커를 어떻게 사용하나요?**
A: 손패에 있는 조커를 원하는 위치로 드래그합니다. 필드의 조커를 내 손패의 해당 타일과 교환할 수도 있습니다.
